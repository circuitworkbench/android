SELECTING A SINGLE COMPONENT OR WIRE
Tap a component or wire in Select mode (the arrow tool). The selected item highlights and its properties appear in the Properties panel. Tap empty canvas to deselect.

MARQUEE SELECT
Tap the Marquee Select tool in the palette (rectangle icon), then drag a rectangle across the canvas. All components fully inside the rectangle are selected when you release. Use this to move, rotate, or delete a group of components at once.

DESELECTING
Tap empty canvas to deselect all. Tapping a different component while one is selected switches the selection to the new component.

WHAT YOU CAN DO WITH A SELECTION
- Move: Drag the selected component or group to a new position.
- Rotate: Tap the Rotate icon in the toolbar to rotate 90° clockwise.
- Mirror: Tap the Mirror icon to flip horizontally.
- Delete: Tap the Delete icon or press Back.
- Edit properties: The Properties panel shows editable fields for the selected component.

SELECTING WIRES
Tap a wire segment to select it. The segment highlights and the Properties panel shows the Node Name field for that net. You can delete individual wire segments without affecting other segments on the same net.
