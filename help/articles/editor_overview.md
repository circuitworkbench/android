The schematic editor has four main areas:

TOOLBAR
At the top of the screen. Contains the circuit name, Save, Undo, Redo, Run Simulation, and the overflow menu (three dots) for less frequent actions like Save As, Rename, and Delete.

CANVAS
The main drawing area. This is where you place components, draw wires, and arrange your schematic. The canvas is infinite in all directions — pan and zoom freely. Components snap to the grid during placement and movement.

PALETTE
A scrollable panel containing all available component types organised by category: Tools, Elementary Devices, Independent Sources, Dependent Sources, Semiconductors, Transformers, Transmission Lines, and Subcircuits. Tap an item to arm the placement tool for that component type.

BOTTOM PANEL
A tabbed panel at the bottom of the screen with three tabs:

- Properties: Shows the editable properties of the currently selected component or wire. Tap any component on the canvas to see its properties here.
- Netlist: Shows the SPICE netlist that will be sent to the simulator. Updates automatically as you edit the schematic.
- Problems: Shows validation errors that would prevent a simulation from running, such as unconnected pins or missing model assignments.

The bottom panel can be dragged up to expand it for more reading space and dragged back down when you need more canvas area.
