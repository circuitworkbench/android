The Properties panel is the bottom tab labelled Properties. It shows the editable parameters of the currently selected component or wire.

OPENING THE PROPERTIES PANEL
Tap any component or wire on the canvas. The Properties panel opens automatically showing that component's fields. If the panel is collapsed, tap the Properties tab to expand it.

PANEL CONTENTS
At the top of the panel is the component type label (e.g. Resistor, NPN BJT, Voltage Source). Below it are the editable fields specific to that component type.

Fields come in three types:
- Text fields: Editable values you type directly (resistance, capacitance, DC voltage, etc.)
- Toggle fields: Boolean on/off switches (e.g. the OFF flag for semiconductors)
- Reference fields: Tap to open a picker (e.g. the Model field for semiconductors)

KEYBOARD BEHAVIOUR
Tapping a text field opens the on-screen keyboard. The Properties panel scrolls to keep the focused field visible above the keyboard. Tap elsewhere or press Done on the keyboard to commit the value.

APPLYING CHANGES
Changes are applied immediately when you move focus away from a field (by tapping another field or tapping the canvas). There is no separate Save or Apply button within the Properties panel. Each change is added to the undo history individually.

WIRE PROPERTIES
Selecting a wire shows the Node Name field, which lets you assign a human-readable name to that net. See Net Names for details.
