CircuitWorkbench is a SPICE schematic editor and simulator for Android. It lets you draw electronic circuits using a touch-friendly canvas and simulate them using the ngspice engine — the same proven simulation core used by professional tools on the desktop.

You can place components from the palette, connect them with wires, assign SPICE models to semiconductors, and run operating point, AC, DC sweep, or transient analyses — all on your phone or tablet.

Circuits are saved as .sch files that contain both the schematic layout and the circuit netlist. The netlist is generated automatically from your schematic every time you run a simulation, so there is no separate netlist editing step required.

CircuitWorkbench supports both IEC and ANSI symbol styles, and includes a model library system that lets you import and reuse SPICE models across circuits.
