By default, tapping a palette item arms the tool for a single placement. After you tap the canvas to place the component, the tool automatically disarms and the editor returns to Select mode.

ENABLING REPEAT MODE
Long-press a palette item instead of tapping it to enter Repeat mode. The placement banner shows a repeat indicator. Now every tap on the canvas places another component of the same type without disarming — useful when you need to place many resistors, capacitors, or other components in sequence.

EXITING REPEAT MODE
Tap the Select tool (arrow icon) in the palette, or press the Back button. The tool disarms and the editor returns to Select mode.

Repeat mode is remembered per palette item for the current editor session. If you switch to a different component type, the new component uses single-placement mode by default unless you long-press it too.
