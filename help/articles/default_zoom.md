The Default Zoom setting controls the zoom level the schematic editor uses when you first open a circuit and when you double-tap the canvas to reset zoom.

ADJUSTING DEFAULT ZOOM
Go to Settings and use the Default Zoom slider. The value is a percentage from 50% (zoomed out) to 200% (zoomed in). The default is 100%.

Choose a default zoom level that shows a comfortable working area for the size of circuits you typically build. A lower value shows more of the schematic at once; a higher value shows component symbols at a larger size which can make placement easier on smaller screens.

RESETTING ZOOM IN THE EDITOR
Double-tap empty canvas in the editor to jump immediately to the default zoom level at any time. This is useful when you have zoomed in to work on a detail and want to quickly return to your usual working view.

The zoom level you are at when you close the editor is not saved — the editor always opens at the default zoom level.
