PLACING A SUBCIRCUIT INSTANCE
1. Tap Subcircuit in the palette. The placement banner appears.
2. Tap the location on the canvas where you want to place the instance.
3. The Subcircuit Picker opens. It has two tabs:
   - File tab: Templates already saved in this circuit's schematic file.
   - Library tab: Templates saved in your app-wide subcircuit library.
4. Use the search field to find a template by name.
5. Tap a template name to select it. The instance is placed at the location you tapped.

PIN LAYOUT
The instance block renders with port stubs on the left and right sides. Ports are distributed evenly — if there are an odd number of ports, the left side gets one more than the right. Port names are labelled inside the block next to each stub.

WIRING THE INSTANCE
Connect wires to the port stubs as you would to any component pin. Tap the Wire tool, then tap a port stub to start a wire from that port.

TEMPLATE LOCKING
Once an instance is placed, it is permanently bound to the template you selected. You cannot re-assign it to a different template afterwards. If you need a different template, delete the instance and place a new one.

TEMPLATES COPIED FROM THE LIBRARY
If you select a template from the Library tab, a copy is automatically embedded in the circuit's schematic file. Future edits to the library template do not affect this instance.
