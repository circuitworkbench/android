To delete a circuit, long-press its card on the home screen and tap Delete. A confirmation dialog appears before the file is permanently removed.

You can also delete from inside the editor via the overflow menu — tap the three dots and choose Delete Circuit.

Deleted circuits cannot be recovered from within CircuitWorkbench. If you use document storage and your device has a recycle bin or trash folder (e.g. Google Drive trash), the file may be recoverable from there for a limited time.

If you want to remove a circuit from the home screen without deleting it, use your device's file manager to move the .sch file to a different folder that is not your configured document storage location.
