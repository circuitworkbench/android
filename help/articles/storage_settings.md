Storage settings control where CircuitWorkbench saves circuit files and how the app-wide libraries are managed. Access them from Settings under the Storage category.

STORAGE LOCATION
Choose between In-App Storage and Document Storage.

In-App Storage saves circuits to the app's private internal directory, managed entirely within CircuitWorkbench and not visible in the system file manager. This is the default and simplest mode.

Document Storage saves circuits to a folder you choose on device storage or cloud storage. You will be prompted to select a folder the first time you save in this mode. Files in document storage are accessible to other apps and are not deleted if you uninstall CircuitWorkbench.

See In-App Storage vs Document Storage for a full comparison.

MODEL LIBRARY
Tap Model Library to open the app-wide SPICE model library manager. Here you can add, edit, delete, and search models. You can also import models from the CircuitWorkbench cloud repository using the overflow menu.

SUBCIRCUIT LIBRARY
Tap Subcircuit Library to open the app-wide subcircuit template library manager. Similar to the model library — add, edit, delete, search, and import templates.

LIBRARY DATA
Both libraries are stored in a local database on your device. Library data is included in Android's app backup (if enabled in your device settings). Uninstalling the app deletes all library data — export important models and subcircuits to a circuit file before uninstalling if you want to preserve them.
