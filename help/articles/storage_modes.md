CircuitWorkbench supports two storage modes, selectable in Settings under Storage Location.

IN-APP STORAGE

Saves circuits to the app's private internal directory. Files are managed entirely within CircuitWorkbench and are not directly visible in your device's file manager or Files app. This is the default mode and the simplest to use. Circuits stored here are backed up with the rest of your app data if you use Google backup.

Uninstalling CircuitWorkbench will delete all circuits stored in app-private storage. Export important files before uninstalling.

DOCUMENT STORAGE

Saves circuits to a folder of your choosing on device storage or cloud storage such as Google Drive. You will be prompted to choose a folder the first time you save in this mode.

Files saved to document storage are visible to other apps, can be shared directly via your file manager, and persist after the app is uninstalled. This mode is recommended if you want to back up circuits manually, share them with a desktop SPICE tool, or work across devices via cloud sync.

SWITCHING MODES

Changing the storage location in Settings affects where new circuits are created. Existing circuits remain in their original location. You can move a circuit by opening it, using Save As to save a copy to the new location, and then deleting the original.
