When a placement tool is armed, a banner appears near the top of the canvas showing the name of the component ready to be placed. For example: "Place Resistor" or "Place NPN BJT".

PLACING THE COMPONENT
Tap anywhere on the canvas to place the component at that location. The component snaps to the nearest grid intersection. For most components, the component is centred on the tap point.

CANCELLING PLACEMENT
Tap the X button in the placement banner, tap the Select tool in the palette, or press the Back button. The tool disarms without placing anything and the editor returns to Select mode.

DRAGGING WHILE ARMED
If you drag on the canvas while a placement tool is armed, the canvas pans instead of placing. This lets you navigate to the area where you want to place the component without accidentally cancelling the tool or placing it in the wrong spot.

SUBCIRCUIT PLACEMENT
The Subcircuit placement tool opens the Subcircuit Picker instead of placing directly on tap. Choose a template in the picker, then the instance is placed at the location you originally tapped.
