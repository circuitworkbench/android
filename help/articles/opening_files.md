Tap any circuit card on the home screen to open it in the schematic editor.

If you use document storage, circuits saved to your chosen folder appear on the home screen automatically. If a file was created outside CircuitWorkbench (for example, copied from a desktop tool or received as an attachment), place it in the folder you configured as your document storage location and it will appear on the home screen on the next refresh.

The home screen refreshes automatically when you return to it from the editor or from Settings. Pull down to refresh manually if a newly added file does not appear immediately.

OPENING .SCH FILES FROM OTHER APPS

If you receive a .sch file via email, messaging, or a file manager, tap the file and choose CircuitWorkbench as the app to open it. The circuit opens directly in the editor.
