The grid helps you align components and wires precisely on the canvas. Grid settings are accessible from Settings → Grid Settings.

GRID VISIBILITY
Toggle the grid on or off. When off, the dot pattern is hidden but snap-to-grid still applies during component placement and movement.

GRID SPACING
Sets the distance between grid points in schematic units. The default spacing works well for most circuits. Increase the spacing if you prefer wider component separation; decrease it for denser layouts.

SNAP TO GRID
When enabled (the default), components and wire endpoints snap to the nearest grid intersection during placement, movement, and wire drawing. This ensures components align neatly and wires connect cleanly to pins.

Disabling snap allows free-form positioning but makes it harder to connect wires to pins precisely. Snap is recommended for all but the most unusual layout requirements.

TIPS
- If a wire appears to connect to a pin visually but does not connect electrically, the wire endpoint may be slightly off-grid. Enable snap and re-draw the wire to ensure it lands exactly on the pin location.
- Zooming in makes it easier to see grid dots and hit exact pin positions, especially for components with closely-spaced pins.
