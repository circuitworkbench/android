Save As creates a copy of the current circuit under a new name, leaving the original file unchanged.

To use Save As:
1. Open the editor overflow menu (three dots in the top-right corner of the toolbar).
2. Tap Save As.
3. Enter the new circuit name and confirm.

The copy is saved immediately and becomes the active circuit in the editor. The original file is untouched and still appears on the home screen under its original name.

Save As is useful for:
- Creating a variant of a circuit without losing the original
- Duplicating a template circuit before modifying it
- Moving a circuit to a different storage location (save the copy, then delete the original)
